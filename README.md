# 📚 ADHD 친화적 학습 스케줄러

예비중학생을 위한 방학 학습 관리 앱입니다.

## 🚀 빠른 시작

### 1. 로컬에서 실행하기
```bash
# Node.js 설치 필요 (https://nodejs.org)
npm install
npm run dev
```
브라우저에서 `http://localhost:5173` 접속

### 2. Vercel로 배포하기 (무료)
1. GitHub에 이 폴더를 업로드
2. vercel.com에서 GitHub 연결
3. 자동 배포 완료!

## 📱 모바일 앱처럼 사용하기

배포 후 스마트폰에서:
- **Android**: Chrome → 메뉴(⋮) → "홈 화면에 추가"
- **iOS**: Safari → 공유(↑) → "홈 화면에 추가"

## ✨ 주요 기능

- 📖 과목별 진도 관리 (국어/수학/영어/사회)
- ⏱️ 시각적 타이머 (원형 진행바)
- 🎮 게임/유튜브 시간 포인트제
- 📊 주간 통계 그래프
- 🎉 완료 시 축하 애니메이션

## 📝 커스터마이징

`src/App.jsx` 파일에서 수정:
- `WEEKDAY_SUBJECTS`: 과목 목록
- `PAGES_PER_SUBJECT`: 과목당 페이지 수
- 게임/유튜브 시간 계산 비율
